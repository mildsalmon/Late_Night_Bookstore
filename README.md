# Late Night Bookstore

내가 발췌한 내용들을 정리하는 나만의 심야 책방.

주로 알람이오는 블로그에서 투자와 관련된 글들을 많이 발행하십니다.  
그렇지만, 투자라는 키워드를 **자신이 중요하게 생각하는 키워드**로 대체해 본다면, 충분히 도움이 되는 인사이트를 얻으리라 믿습니다.
